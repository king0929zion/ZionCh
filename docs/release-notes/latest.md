## ZionChat v0.36.0

- 继续重构 `ZiCode`：主功能入口与 `ZiCode 设置` 入口彻底分离，模块身份统一改为 ZiCode 专属图标，不再混用 GitHub 图标。
- ZiCode 界面继续向主设置页靠齐，纯灰白黑体系保持不变，主灰固定为设置页同款灰色，并进一步压平页面层级，减少仓库页和会话页的套娃结构。
- 仓库列表继续简化：仓库卡片只保留左侧公开性黑底标签与更大的仓库名，移除用户名等多余信息，整体更轻更干净。
- 会话页交互重做：顶部不再显示多余提示与横排会话项，多会话改为抽屉式切换；ZiCode 专用输入框独立实现主对话同款样式，并修正底部贴边与键盘跟随行为。
- 默认模型能力闭环补齐：默认模型页新增 ZiCode Agent 选项，ZiCode 运行时强制使用单独配置的 ZiCode 模型，不再回退到普通聊天模型。
- GitHub Agent tools 继续扩充，现已覆盖账号校验、仓库读取与创建、目录树/文件预览/文件写入、分支读取与创建、提交历史、Actions 扫描与触发、运行控制、Release 与 Pages 状态等能力。
- 继续优化远端构建速度：保留 Kotlin 增量缓存与并发优化设置，减少版本号更新时带来的无效缓存失效。
